# Repo Triage

Use this skill when the goal is to inspect a codebase, gather the minimum required context, and propose or execute a safe triage plan.

Guidelines:

- Start with read-only inspection.
- Prefer listing and schema discovery before writes.
- Use explicit tool reasons.
- Stop and request approval if the next step becomes destructive or high-risk.
- Summaries should separate findings, risks, and concrete next actions.
